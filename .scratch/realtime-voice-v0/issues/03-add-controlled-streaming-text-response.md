# 接入受控的流式文字回复

Status: `ready-for-human`
Type: `HITL`

## What to build

将用户的一轮最终转写发送给一个可替换的流式对话模型，并在同一页面持续显示 AI 的增量文字回复。用户需要在实现前确认本轮使用本地模型还是中国大陆可访问的云端模型，以及所需的凭证或运行资源。

## Acceptance criteria

- [ ] 明确并记录首轮对话模型实现方式：本地模型或国内云端流式模型。
- [ ] 模型接入通过独立适配器完成，后续更换实现不改变会话流程。
- [ ] 用户完成一轮语音输入后，页面能够流式展示 AI 文字回复。
- [ ] 支持多轮临时上下文，结束会话或刷新页面后不保留内容。
- [ ] AI 回复受到基础边界约束，例如回复简短、适合口语交互且不生成过长独白。
